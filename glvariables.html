Local and Global Variables

What is the value of x at the end?
x = 3
def mystery(y):
 x = y + 2
 return x
x = mystery(x)

(a) 2

(b) 3

(c) 5

(d) There would be an error

Correct!

The return statement reassigns x to a new value. Passing x into the mystery function adds 2 to x, returning 5.

There is no error because all of these variables are clearly defined and able to communicate with each other.
